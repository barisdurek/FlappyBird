# Flappy Bird with SDL Library
A version of Flappy Bird developed using the SDL library.

### Downloading the libraries:
```
sudo apt install libsdl2-dev libsdl2-image-dev libsdl2-ttf-dev libsdl2-mixer-dev
```

### Debugging:
```
make
```

### Running the game:
```
./flappy
```
